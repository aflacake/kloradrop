Tentang KloraDrop

KloraDrop adalah aplikasi-web untuk membantumu menjaga asupan air minum, mengingatkan waktu minum dan menyimpan progres harianmu. Semua data tersimpan di perangkatmu tanpa memerlukan internet!

KloraDrop, aplikasi yang dikembangkan bertujuan seseorang tidak melupakan jam air minumnya, selalu tepat, disiplin, dan target air putih terpenuhi.

KloraDrop diambil dari nama "Klora" kesan karakter pintar & terpecaya, aplikasi yang memberikan banyak tips edukasi kesehatan. Karena maskotnya tunas hijau, namanya mengandung 'O' secara psikologis huruf tersebut terdengar lebih "bulat" dan "imut" di telinga, cocok untuk makhluk kecil seperti kecambah.
Nama Klora kesannya organik, sehat, dan sangat identik dengan tunas hijau (karena klorofil). Untuk memberikan kesan "disiplin" dan "inum air putih".

"Drop" artinya tetesan. Sangat visual, seperti embun di atas tunas hijau. Cocok untuk maskot kecambah.

Informasi Penyimpanan Data

Aplikasi ini mendukung penyimpanan lokal pada:

  • Google Chrome 49+
  • Mozilla Firefox 57+
  • Microsoft Edge 79+
  • Safari iOS/macOS 11+
  • Opera 36+
  • Android Browser Chromium-based

Jika kamu membersihkan data browser, progres minum akan terhapus.

Kelola Data

Ekspor Data

Kamu bisa menyimpan data minum ke berkas (.json) agar bisa dipindahkan ke perangkat lain atau disimpan sebagai cadangan. Berkas ini berisi:

  • Total minum harian (maks 8 gelas per hari)
  • Total minum mingguan (maks 56 gelas per minggu)

Lokasi File Ekspor

File hasil ekspor akan tersimpan di folder:

  • HP Android/iOS: folder Unduhan atau lokasi yang dipilih melalui dialog simpan
  • Windows dan macOS: folder Downloads/Unduhan pengguna
  • Safari iOS: pengguna diminta memilih lokasi penyimpanan

Data KloraDrop disimpan secara lokal di browsermu (Chrome, Edge, Firefox, Safari, dll) bukan di server. Jika kamu menghapus data browser atau memakai mode penyamaran, data dapat hilang.
